# Milestone 0.2 - 功能增强与 PC 端适配

## 1. 目标
基于 Milestone 0.1 完成的核心记账功能，进一步增强数据统计能力，优化 UI/UX 体验，并提供完整的 PC 端（Tauri）开发环境支持。

## 2. 需求详情

### 2.1 环境与运行 (Tauri)
- **文档输出**：新增 `docs/tauri-setup.md`。
- **内容要求**：
    - Windows 环境下 Rust 安装指南。
    - Tauri CLI 安装与环境配置。
    - 常用开发命令 (`npm run tauri dev`)。

### 2.2 统计功能增强
- **时间维度**：增加“日、周、月、年、全部”五个维度。
- **快捷操作**：新增 5 个快捷按钮（今天、昨天、上周、上月、本年度）。
- **默认状态**：默认选中“本月”。

### 2.3 UI/UX 与样式架构
- **样式重构**：引入 CSS 变量 (`:root`) 定义主题色、圆角、间距。
- **交互优化**：
    - 全局添加 `cursor: pointer`。
    - 侧边栏 `z-index` 层级修复。
- **国际化**：全站文案中文化（替换 "Personal Finance" 等）。
- **组件替换**：移除所有 `alert()`，使用自定义 Modal/Popover 组件。

### 2.4 数据导入导出重构
- **入口合并**：将“导入”与“导出”合并为“数据管理”按钮。
- **TXT 导出**：实现 TXT 格式导出功能，使用 `\u0001` 分隔符，格式需与导入格式兼容。
- **功能补全**：确保导入导出数据格式一致。
