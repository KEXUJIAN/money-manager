## 关键决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| 状态管理 | useLiveQuery | Dexie 原生响应式，避免额外依赖 |
| 内置分类 | isBuiltin 字段 | 用户提供的 58 个分类不可删除，自定义分类可增删 |
| 历史数据分隔符 | `\u0001` (SOH) | 用户原有数据格式 |
| 日期格式 | yyyy-MM-dd | 修正了错误的 `PHP` 格式字符串 |

## 已知限制

- 浏览器 IndexedDB 存储受浏览器配额限制（通常 > 1GB）
- 暗色模式 CSS 变量已定义，但未做完整适配
- Recharts 与 React 19 存在部分类型不兼容，使用 `any` 绕过
