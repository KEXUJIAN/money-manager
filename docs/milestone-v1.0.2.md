# Milestone v1.0.2

## 目标
实现全局悬浮“记一笔”按钮，并支持分类名称的中文拼音搜索。

## 任务执行记录

### 1. 全局记账悬浮按钮
- 移除了由于早期独立组件重构保留在 `Home` 级的局部记账按钮。
- 修改并升级 `AddTransactionButton` 样式，在 `AppLayout` 中注册为全局 Float Action Button (FAB)。
- 对不同系统端口（移动端/桌面端）均保持位置的一致性和交互的微动效反馈。

### 2. 分类名称拼音搜索
- 根据技术选型决议，引入轻量级库 `pinyin-match` 到当前环境。
- 在通用的 `SearchableSelect` 搜索弹窗组件的过滤项中注入了 `PinyinMatch.match` 逻辑。
- 实现了同时兼容汉字模糊查询与拼音/首字母直接匹配的功能，解决了自定义分类无法被拼音捕捉的痛点。

## 测试与交付
执行了静态类型校验（tsc）与生产环境（vite build）构建，结果无误。所有任务列表中的功能开发及验证均顺利结束。
